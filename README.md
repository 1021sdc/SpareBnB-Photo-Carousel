## Usage

```bash
# Install dependencies
$ npm install

# Create CSV file to insert into PostgresSQL
$ npm run seedFile

# Bundle files with webpack
$ npm run build

# Run the app
$ npm run start
```

> Copy CSV file into into PostgresSQL
